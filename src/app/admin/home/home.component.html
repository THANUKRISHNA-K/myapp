USER MODULE

The user module allows users to register, log in, and log out. Each user is assigned one or more roles. By default there are three roles: anonymous (a user who has not logged in) and authenticated (a user who is registered), and administrator (a signed in user who will be assigned site administrator permissions). Once got loggedin to the administrator component the admin is allowed to change the user module like manually creating new users, removing the existing users and to update user details.
